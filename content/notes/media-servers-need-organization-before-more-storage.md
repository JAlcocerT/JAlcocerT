---
title: "Media servers need organization before more storage"
date: 2026-05-12
tags: ["self-hosting", "media", "homelab"]
description: "A media server becomes useful when files, metadata, access, and backups are organized."
---

More storage does not make a media server better by itself.

The hard part is organization: naming, metadata, transcoding, access, backups, indexes, permissions, sync, and device workflows. Without that, the server becomes a larger pile of files.

Media infrastructure is mostly information architecture.
