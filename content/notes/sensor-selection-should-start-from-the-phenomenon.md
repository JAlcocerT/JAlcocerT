title: "Sensor selection should start from the phenomenon"
date: 2026-10-25

description: "Choosing sensors starts with the physical variable, required accuracy, environment, and decision to be made."
---

Sensor selection should start with the phenomenon being measured.

Temperature, humidity, pressure, light, motion, current, and distance each impose accuracy, placement, calibration, sampling, and durability constraints. The sensor is only useful if it measures the variable well enough for the intended decision.

Measurement starts from purpose.
