---
title: "Cameras become useful when detection creates an action"
date: 2026-08-01
tags: ["computer-vision", "homelab", "nvr"]
description: "Object detection matters when it changes alerts, automation, storage, or decisions."
---

A camera system is not smarter just because it detects objects.

Detection becomes useful when it changes an action: alert, record, ignore, classify, automate, summarize, or trigger another system. Otherwise it is just metadata.

The value of computer vision is downstream action.
