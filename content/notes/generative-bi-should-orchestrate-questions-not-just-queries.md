---
title: "Generative BI should orchestrate questions, not just queries"
date: 2026-01-16
tags: ["gen-bi", "analytics", "agents"]
description: "A Gen-BI system should help structure the analytical question before generating SQL."
---

Generative BI is more than text-to-SQL.

A good system helps decide which question is being asked, which metric definition applies, which filters are valid, and whether the answer should be a table, explanation, chart, or follow-up investigation.

The agentic layer is useful when it improves analytical framing, not when it merely hides SQL.
