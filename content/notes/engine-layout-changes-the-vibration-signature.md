Engine architecture changes the vibration signature.

Flat-plane, cross-plane, inline, V, and boxer layouts distribute inertial and combustion events differently. The result is not only a sound difference; it changes force orders, moments, packaging, and mount strategy.

The layout is part of the signal source.
